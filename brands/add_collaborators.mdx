## What is a collaborator?

Collaborators contribute video content to Loops. Every collaborator must submit a request to collaborate to a moderator, who can approve or deny the request.

## How to add and manage collaborators?

### How to select and add collaborators for your brand
1. Select the Community that you would like to add collaborators. 
2. Go to the Loops tab of your desired Community and select the specific Loop(s) you wish to add a collaborator.
3. Click the tab labeled “Collaborators” 
4. Tap the plus (+) icon select “Add Collaborators” 

### Manage content created by collaborators

Any content that is created by collaborators can be approved or denied by the moderator within the Brand Control Center.
