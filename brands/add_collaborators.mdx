## What are Collaborators in Loops?
Collaborators can contribute to any Loop – One can request this, in which a Moderator can approve/deny the request. 

## Who could be Collaborators for my brand and how can I add them?
Select the desired Community you would like to add Collaborators to. Once this brings you to the Loops tab, select the specific Loop(s) you wish to add Collaborators. Click on ‘Collaborators’ (second tab).  Tap the + icon where it states “Add Collaborators.” 

## How to manage content that is created by Collaborators?
Any content that is created by Collaborators can be approved/denied within the Brand Control Center.
