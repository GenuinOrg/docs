## How to embed your Community?

### How to embed the Community on your website

Once you’ve decided to whitelable your Community, you must embed it on your website. There are two ways to do this: 

If you are an existing user you can go to the settings module and set up an embed from there. Follow these steps:

1. Go to the embed section and add the embed name.
2. Select embed content and select embed type (Loop, Community, Single Video)
3. Customize specifics from the side panel.
4. Preview the embed view and generate the code. You can see the generated code for web/iOS/Android or you can use a QR code.
5. Paste the code in your desired location across your website (e.g. home page, community page, etc.) 
6. Revisit the embed module and check the embed analytics on the landing page 

For first-time users: 
1. Create your user and brand profiles.
2. Add your brand elements.
3. Create your first community.
4. Go to the embed module and follow the steps above.
		
### What are embed types?	
Embed type refers to the variety of content formats that you can integrate across your website. There are 6 embed types.
1. Brand Feed
2. Community Feed
3. Community Page
4. Loop Feed
5. Loop Page
6. Post
