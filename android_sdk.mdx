## Installation

<CodeGroup>

```kotline Kotline Using build.gradle.kts
// 1. Need to add below code in project level settings.gradle.kts
dependencyResolutionManagement {
  repositoriesMode.set(RepositoriesMode.FAIL_ON_PROJECT_REPOS)
  repositories {
    google()
    mavenCentral()
    jcenter()
    maven {
      setUrl("https://jitpack.io")
    } 
  }
}

// 2. Need to add dependency in app level build.gradle.kts
implementation("org.bitbucket.genuindev:genuin_android_sdk:1.0.0@aar{
   isTransitive=true
}

```
