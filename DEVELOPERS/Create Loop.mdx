# Create Loop

The **Create Loop** API enables quick setup of collaborative spaces called Loops, where collection of videos organized around a specific theme or topic.


**URL**  :  `/api/v3/conversation/create`

**Method**  :  `POST`

**Auth required**  : YES

**Mandatory Fields**: type, Community_id

## Bad Request

**Condition**: If the request contains errors or invalid data, resulting in the loop not being created.

**Code**: `400`

**Content**
```
{}
```

## Internal Server Error

**Condition**: If an unexpected issue occurs on the server, an `Internal Server Error` is returned.

**Code**: `500`

**Content**
```json
{
  "code": "5026",
  "message": "Could not find the video.",
  "data": {}
}
```
